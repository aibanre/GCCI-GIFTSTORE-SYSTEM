ARTIFICIAL INTELLIGENCE INTEGRATION
------------------------------------
AI Provider: Google Gemini AI
API Endpoint: https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent
Model: gemini-2.0-flash
Configuration: Environment variables (AI_API_URL, AI_API_KEY)

AI Features:
  1. Dynamic Product Carousel Recommendations
     - Analyzes product inventory
     - Generates contextual product showcases
     - Multiple recommendation strategies (new stock, trending, low stock, seasonal)
     - Response format: JSON with title and product recommendations
  
  2. AI Analysis Logging
     - Tracks all AI requests and responses
     - Stores prompts, responses, and metadata
     - Provider tracking (Gemini)
     - Performance monitoring

AI Implementation Details:
  - Server-side API key management (secure)
  - Configurable via environment variables or config.json
  - Fallback handling for API failures
  - Response parsing for Google Gemini format
  - Database logging of AI interactions
